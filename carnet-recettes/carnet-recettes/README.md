# 📖 Mon Carnet de Recettes

Application PWA de gestion de recettes avec adaptation des portions.

## 🚀 Déploiement sur Vercel (gratuit, 5 minutes)

### Étape 1 — Créer un compte Vercel
👉 https://vercel.com/signup (gratuit, connectez-vous avec GitHub)

### Étape 2 — Créer un repo GitHub
1. Allez sur https://github.com/new
2. Nom : `carnet-recettes`
3. Cliquez "Create repository"
4. Uploadez tous les fichiers de ce zip dans le repo

### Étape 3 — Déployer sur Vercel
1. Allez sur https://vercel.com/new
2. Importez votre repo GitHub `carnet-recettes`
3. Laissez les paramètres par défaut (Vercel détecte Vite automatiquement)
4. Cliquez **Deploy** → votre app est en ligne en ~1 minute !

### Étape 4 — Installer sur votre téléphone

**iPhone (Safari) :**
1. Ouvrez votre URL Vercel dans Safari
2. Appuyez sur le bouton Partager (carré + flèche)
3. → "Sur l'écran d'accueil"
4. Appuyez "Ajouter"

**Android (Chrome) :**
1. Ouvrez votre URL Vercel dans Chrome
2. Menu ⋮ → "Ajouter à l'écran d'accueil"
